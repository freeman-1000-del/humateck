유튜브등록 FAQ
1. 문) 번역 등록 후 스튜디오에서 자막을 열어보니 번역은 되었는데, 설명문칸이 비어 있습니다.
답) 유튜브 스튜디오 메뉴 상단의 '세부정보'를 눌러 스튜디오 우측 '자막'칸의 '동영상 언어 설정'을 확인하십시오. 작성하신 언어와 '동영상 언어 설정'이 일치하지 않으면 등록 시 유튜브의 거부로 설명란이 공백으로 남게 됩니다.
2. 문) 유튜브 자막 등록하기를 누르니 할당량 초과 메시지가 뜹니다.
답) 유튜브 자막등록 일일 API 무료할당량이 초과된 경우입니다. 다음날 할당량이 다시 충전될 때까지 기다렸다가 다시 시도하시기 바랍니다.
3. 문) '유튜브 영상 주소에서 videoId를 찾지 못했습니다.' 메시지가 뜹니다.
답) 유튜브 주소를 잘못 입력하신 경우입니다. 반드시 유튜브 동영상(스튜디오 ×)의 주소를 그대로 복사하여 붙여 넣어 주시기 바랍니다.
4. 문) 구글 OAuth 설정이 무엇인가요? 사용시마다 설정해야 하나요?
답) 내 유튜브 채널에 자막을 등록해도 좋다고 구글에 허락받는 연결 과정입니다. 처음 1회만 설정하시면 됩니다.
5. 문) 여러 채널을 운영 중입니다. OAuth 설정을 단 한번만 받아도 되나요?
답) 구글의 정책상, 보유하신 채널당 한 번씩의 설정이 필요합니다.
6. 문) YouTube Data API 할당량이란 무엇인가요?
답) 유튜브가 하루 제한량을 설정하는데 이를 API 할당량이라고 합니다. 기본적으로 일일 10,000단위가 주어지며, 태평양 표준시(PT) 자정에 재설정됩니다.

  1. '보류사유복사' 버튼을 눌러 내용을 복사해 주세요.
  2. 번역을 담당한 해당 제미나이 대화창에 그대로 붙여넣어 주세요.
  3. 제미나이의 답변 내용을 확인하시고, 이상 유무를 검수해 주세요.
  4. 이상이 없을 경우 수정된 내용을 복사해 주세요.
  5. '보류언어 재배송' 버튼을 눌러 붙여넣어 주세요.
  6. 최종 확인 후 배송을 진행해 주세요.

⚠️ 필수! 제출 전 확인사항

자막등록하시려는 채널의 동영상언어가 작성하신 제목과 설명문의 언어와 일치되도록 설정하셨나요?

이것이 일치하지 않은 상태로 번역물을 생성하여 등록 시, 유튜브 시스템 정책상 '원문 없음'으로 처리되어 다음과 같은 번거로운 일이 생깁니다.

  1. 다국어 번역은 모두 입력되지만, 원문 '설명문'칸이 공백으로 남게 되며 '자막기본설정은 영문으로 박제'됩니다.
  2. 상기 설명문의 공백을 메우려면, 부득이 원문을 '영문'으로 재번역하여 고객님의 채널 원설명문칸에 한번만 더 '복사 붙여넣기'를 하셔야 합니다.
