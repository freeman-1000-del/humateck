FAQ

1. 구글 OAuth 설정이 무엇인가요? 사용시마다 설정해야 하나요?
내 유튜브 채널에 자막을 등록해도 좋다고 구글에 허락받는 연결 과정입니다. 처음 1회만 설정하시면 됩니다.
2. 여러 채널을 운영 중입니다. OAuth 설정을 단 한번만 받아도 되나요?
구글의 정책상, 보유하신 채널당 한 번씩의 설정이 필요합니다.
3. 번역 등록 후 설명문칸이 비어 있습니다.
유튜브 스튜디오 '세부정보' → '자막' → '동영상 언어 설정'을 확인하십시오. 작성하신 언어와 일치하지 않으면 설명란이 공백으로 남게 됩니다.
4. 유튜브 자막 등록하기를 누르니 할당량 초과 메시지가 뜹니다.
유튜브 자막등록 일일 API 무료할당량이 초과된 경우입니다. 다음날 할당량이 재충전될 때까지 기다렸다가 다시 시도하시기 바랍니다.
5. '유튜브 영상 주소에서 videoId를 찾지 못했습니다.' 메시지가 뜹니다.
유튜브 주소를 잘못 입력하신 경우입니다. 반드시 유튜브 동영상(스튜디오 ×)의 주소를 그대로 복사하여 붙여 넣어 주시기 바랍니다.
6. YouTube Data API 할당량이란 무엇인가요?
유튜브가 하루 제한량을 설정하는데 이를 API 할당량이라고 합니다. 기본적으로 일일 10,000단위가 주어지며, 태평양 표준시(PT) 자정에 재설정됩니다.
