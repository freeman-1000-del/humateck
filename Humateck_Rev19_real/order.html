유튜브 등록관련 공지사항

당사는 고객님의 번역 승인본에 대해 어떠한 검수나 차단도 하지 않으며, 승인된 내용 그대로 발송합니다.

모든 번역문은 유튜브 채널에 즉시 등록되며, 접수측이 실제로 거부한 언어는 보류 언어로 분리 표시됩니다.

유튜브등록 FAQ
1. 문) 번역 등록 후 스튜디오에서 자막을 열어보니 번역은 되었는데, 설명문칸이 비어 있습니다.
답) 유튜브 자막은, 영어가 아닌 다국어 원문으로 번역하여 다국어로 등록시 기본 언어설정이 영어(미국어)로 자동전환됩니다. 따라서 고객님의 모국어 역시 외국어로 번역 처리되어 설명칸이 공백으로 남게 되며, 이는 유튜브의 다국어 자막 표시시스템상 불가피한 일입니다. 필요시 고객님이 작성한 원문을 공백에 다시 복붙하셔서 사용하시기 바랍니다.
2. 문) 유튜브 자막 등록하기를 누르니 할당량 초과 메시지가 뜹니다.
답) 유튜브 자막등록 일일 API 무료할당량이 초과된 경우입니다. 다음날 할당량이 다시 충전될 때까지 기다렸다가 다시 시도하시기 바랍니다.
3. 문) '유튜브 영상 주소에서 videoId를 찾지 못했습니다.'라는 메시지가 뜹니다.
답) 유튜브 주소를 잘못 입력하신 경우입니다. 반드시 유튜브 실제동영상(스튜디오 ×)의 주소를 그대로 복사하여 붙여 넣어 주시기 바랍니다.
4. 문) 구글 OAuth 설정이 무엇인가요? 사용시마다 설정해야 하나요?
답) 내 유튜브 채널에 자막을 등록해도 좋다고 구글에 허락받는 연결 과정입니다. 처음 1회만 설정하시면 됩니다.
5. 문) 여러 채널을 운영 중입니다. OAuth 설정을 단 한번만 받아도 되나요?
답) 구글의 정책상, 보유하신 채널당 한 번씩의 설정이 필요합니다.
6. 문) YouTube Data API 할당량이란 무엇인가요?
답) 유튜브는 자막테이터 사용에 대해 채널당 하루 제한량을 설정하는데, 이를 API 할당량이라고 합니다. 기본적으로 일일 10,000단위가 주어지며, 태평양 표준시(PT) 자정에 재설정됩니다.

⚠️ 필수! 제출 전 확인사항

자막등록하시려는 채널의 동영상언어가 영문(미국어)이 아니신가요?

고객님의 모국어로 번역물을 생성하여 등록 시, 다국어 번역은 모두 입력되지만, 원문 설명문 칸이 공백으로 남게 되며 자막기본설정은 영문(미국)으로 박제됩니다.

하지만 이는 유튜브 다국어자막 등록시스템상 불가피한 현상이며 번역품질에 영향을 미치는건 아닙니다. 필요시 모국어 채널에 제미나이의 번역 대신 고객님이 직접 작성하신 원문 설명문을 넣어서 쓰실 수 있습니다.
